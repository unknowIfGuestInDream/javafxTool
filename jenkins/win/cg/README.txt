# windows下运行jar

要求JAVA_HOME 版本在17+
或者在当前文件夹下存在java jre

将 console.bat 和 cgTool.bat 与javafxTool-cg.jar 放置同一路径下,
运行cgTool.bat, 启动程序

# Run the jar on Windows

Requirements JAVA_HOME Version 17+
Or there is a folder 'jre' under the current folder

Place console.bat and cgTool.bat in the same path as javafxTool-cg.jar,
Run cgTool.bat to start the program

# Windowsでjarを実行する

バージョン 17+ JAVA_HOME要件
または、現在のフォルダの下にjava jreがあります

console.bat と cgTool.bat を javafxTool-cg.jar と同じパスに配置します。
cgTool.bat を実行してプログラムを起動します
