## 介绍

javafx工具包，JDK17 + JavaFX17 + controlsfx 11.x.x + Maven

frame 和 login 是基础模块, 通过java SPI实现可拔插，方便应用模块集成  
demo是提供的示例demo

## 结构

frame: 应用框架  
demo: demo例子  
smc: 个人应用  
login: 登录模块
